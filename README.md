# Star Wars

<h2>Como testar?</h2>

- Baixar o repositório.
- Extraí-lo.
- Abrir a pasta dentro de uma IDE de preferência com Emulador(cel), porém da pra testar no navegador.
- Executar o Pub Get.
- Rodar a aplicação no Emulador ou Navegador.

- ou pode instalar o APK no celular
